# Changelog

Все заметные изменения проекта документируются в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/).

## [1.0.0] - 2025-01

### Добавлено

#### Вкладка "Плеер"
- Воспроизведение исходных аудиофайлов (WAV, MP3)
- Воспроизведение обработанных файлов из папки output
- Разделение на колонки: исходные / обработанные файлы
- Контролы: воспроизведение, пауза, стоп
- Регулировка громкости (слайдер 0-100%)
- Прогресс-бар с перемоткой
- Информация о файле (имя, размер, дата изменения)
- Фильтрация обработанных файлов по исходному

#### Вкладка "Спектр"
- Спектральный анализ исходного файла
- Сравнение спектров нескольких методов
- Выбор исходного файла из уже обработанных
- График амплитуды (дБ) от частоты (Гц)
- Цветовая дифференциация методов
- Автоматическое определение диапазона Y-оси

#### Экспорт в Excel
- Сохранение таблицы результатов в .xlsx
- Автоматическое именование файла с датой
- Форматирование заголовков и данных

#### Матрица влияния настроек
- Таблица методы × метрики
- Качественная оценка влияния параметров (↑↑, ↑, →, ↓, ↓↓)
- Цветовая кодировка (зелёный, серый, красный)
- Переключаемая легенда

#### Пресеты настроек
- "Стандартный" — демонстрация различий методов
- "Без сжатия (identity)" — тест идентичности
- "Быстрый" — оптимизация скорости
- "Качество" — максимальное качество
- "Лучший средний" — баланс качества и скорости

#### Пакетная обработка
- Выбор папки с WAV-файлами
- Рекурсивный поиск файлов
- Подсчёт файлов и жанров
- Прогресс-бар обработки

### Изменено

#### Рефакторинг модуля processing
- Выделен модуль `processing/metrics.py` (~350 строк)
- Переработан `audio_ops.py` (~720 строк)
- Улучшена структура публичного API
- Добавлены экспорты в `__init__.py`

#### Улучшена обработка ошибок
- Заменены silent exception handlers (except: pass)
- Добавлено логирование ошибок
- Диагностика для PyInstaller сборок

#### Пресеты
- Изменён пресет "Стандартный" для демонстрации различий методов
- Режим отбора: none → energy (0.95)

### Техническое

#### Unit-тесты
- Создан `tests/test_metrics.py`
- 19 тестов для всех метрик качества
- Покрытие: временная область, спектральная область, краевые случаи

#### Документация
- Обновлён README.md для версии 1.0
- Создан CHANGELOG.md
- Диаграммы в docs/diagrams/

#### Качество кода
- Добавлены type hints (numpy.typing)
- Улучшена диагностика в hooks и scripts
- Проверка синтаксиса всех файлов

## [0.1.0] - 2024-12

### Добавлено

- Базовая структура проекта
- Методы обработки: FWHT, FFT, DCT, DWT, Huffman-like, Rosenbrock-like
- Метрики качества: SNR, RMSE, SI-SDR, LSD, Spectral Convergence, Centroid Δ, Cosine Similarity
- Вкладка "Таблица" с результатами
- Вкладка "Сравнение" с графиками
- Вкладка "Настройки" с параметрами
- Кодирование в MP3 через FFmpeg
- Сборка .exe через PyInstaller
