# processing

Модуль вычислительной части и кодеков.

Содержимое:
- `audio_ops.py` — пайплайны: FWHT/FFT/DCT/DWT/варианты, расчёт метрик и агрегирование.
- `fwht.py` — чистая математика FWHT и OLA-процедуры.
- `codecs.py` — взаимодействие с FFmpeg/pydub/soundfile; декодирование и кодирование MP3.
- `api.py` — устаревший фасад для обратной совместимости (реэкспорт функций из `audio_ops`).

Зависимости: `numpy`, `soundfile`, `pydub`/`ffmpeg`, стандартные библиотеки.

Использование в системе:
- UI вызывает функции `*_transform_and_mp3(...)` и затем `_compute_metrics_batch(...)` для отображения результатов.
