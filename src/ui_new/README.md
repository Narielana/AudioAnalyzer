# ui_new — Единый модуль UI AudioAnalyzer

## Назначение

Этот модуль является **полной перезаписью** UI для проекта AudioAnalyzer, объединяющей
функционал из старых модулей `ui/`, `ui2/` и `app_ui/` без дублирования кода.

## Структура

```
ui_new/
├── __init__.py           # Экспорты модуля
├── main_window.py        # Главное окно (779 строк)
├── worker.py             # Фоновая обработка (498 строк)
├── constants.py          # Константы UI (125 строк)
├── presets.py            # Пресеты настроек (303 строк)
├── log_handler.py        # Логирование в UI (89 строк)
└── widgets/              # Виджеты (для будущего расширения)
```

**Итого: ~1838 строк** (против ~2500+ в старых модулях)

## Ключевые улучшения

1. **Исправлена ошибка QtLogHandler** — метод `emit` теперь внутри класса
2. **Нет дублирования Worker** — единая реализация
3. **Нет циклических зависимостей** — чёткая иерархия импортов
4. **Объединённый функционал** — лучшее из ui и ui2:
   - Dataclass ResultRow (из ui2)
   - Полная статистика ETA (из ui)
   - Heatmap и подсказки метрик
   - Пресеты настроек

## Использование

```python
# Простой импорт
from ui_new import MainWindow

# Создание окна
window = MainWindow()
window.show()
```

## Совместимость

- Python 3.10+
- PySide6
- Работает как в режиме исходников, так и в PyInstaller
